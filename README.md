# BreathingApp

Wim Hof–style breathing exercise app with guided cycles and custom routines.

## What’s included

- iOS “Add to Home Screen” support (apple-touch-icon + meta)
- Basic PWA manifest
- Accessible dialogs and buttons (ARIA labels, Escape to close)
- Keyboard support (Space/Enter for breathing control)
- Settings persistence (speech volume)

## Run locally

You can open `index.html` directly in your browser, or start a dev server:

```powershell
npm install
npm run dev
```

On iOS Safari, tap Share → Add to Home Screen to install as a standalone app.
